# _Detection_Face

Данная курсовая работа посвящена разработке интеллектуальной системы видеоаналитики для ритейла. Проект реализует каскадную архитектуру нейросетей для автоматизации оценки посещаемости и анализа востребованности товаров среди различных демографических групп.

Система способна работать в сложных сценах: при нестабильном освещении, высокой плотности человекопотока и частичном перекрытии объектов (окклюзиях).

Ключевые возможности
Детекция лиц: Использование современных архитектур YOLO для локализации лиц в реальном времени.

Демографический анализ: Определение пола (классификация) и возраста (регрессия) каждого обнаруженного человека.

Многозадачное обучение: Кастомная архитектура классификатора на базе ResNet18 с разделенными выходными слоями.

Интерактивный интерфейс: Веб-приложение для загрузки изображений и мгновенного получения результатов аналитики.

ехнический стек
Backend: FastAPI (асинхронная обработка запросов).

Computer Vision: - YOLOv8/v11: Детекция лиц (обучение на датасете WIDER FACE).

ResNet18: Профилирование (обучение на датасете UTKFace).

Frontend: HTML5, CSS3, JavaScript (рендер результатов и отрисовка Bounding Boxes).
